# Screenshot proyek

Masukkan screenshot dengan nama berikut:

```text
david-sales.webp
finance-panel.webp
wedding-invitation.webp
home-server.webp
```

Rekomendasi:

- Format WebP
- Rasio 16:10
- Resolusi sekitar 1600 × 1000 piksel
- Ukuran setiap gambar di bawah 400 KB

Gambar akan otomatis menggantikan placeholder proyek. Nama dan lokasi file dapat diubah melalui `config/portfolio.php`.

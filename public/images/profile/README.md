# Foto profil

Masukkan foto profil ke folder ini dengan nama:

```text
david.webp
```

Rekomendasi:

- Format WebP
- Rasio mendekati 4:5 atau persegi
- Lebar 1000–1600 piksel
- Ukuran file di bawah 500 KB

Setelah file tersedia, foto otomatis menggantikan tampilan inisial tanpa mengubah Blade atau JavaScript.
